## Project Description
#### ***The Spark ML library in JAVA is used in this project to classify comments as bullying or non-bullying. A dataset from the Kaggle website is used to train five machine learning models, and the outcomes are compared. The tokenized comments are transformed into feature vectors using the count and TF-IDF vectorizers. Models'Â performance in the test dataset is evaluated using the accuracy metric.***
